# Roll-a-Ball Game

- The player uses the arrow keys to move and roll the ball around the play area  
- The player collects pickups scattered across the scene  
- An enemy moves in the scene and must be avoided  
- If all pickups are collected, the player wins  
- If the player is caught by the enemy, the game is lost
